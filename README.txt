Целевая папка проверки: src/main/java/org/example/hw

Представлено 2 пункта дз:
-дерево
-резервное копирование

В дереве доработан вывод файлов совместно с папками (комментарии оставлены после семинара, для себя)

Резервное копирование имеет 2 способа реализации
(обе копируют результат на уровень src/)

Первый, простой, рекурсивно проходит по содержимому папок
 считывает и сохраняет в целевую

Второй вариант, более сложный, реализован с помощью потока DirectoryStream<Path>
 создает правильную ссылку для новой папки, куда потом копирует текущий файл
 (комментарии - краткое объяснение последовательности)
